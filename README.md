## Quickstart

### Checkout Repo

```
git clone https://github.com/ritvarsbiksis/react-todo-app.git
```
```
cd react-todo-app
```

### Install deps

```
npm install
````

### Run for PRODUCTION

```
npm run start
node index.js
````


**[http://localhost:3033/](http://localhost:3033/)**

### Run for DEV with Webpack watcher

```
npm run dev
````

**[http://localhost:3033/](http://localhost:3033/)**
